python小插件

Jupyter

  • yupyter是一个python的gui小插件，挺好用的，主要是将python的cmd界面搬到网页上，然后在网页上输入代码，调试结果。

  • 安装

  1. python安装成功，环境配置好；
  2. 在cmd上输入代码pip install ipython[all] -i https://pypi.douban.com/simple 下载该插件；
  3. 下载安装成功之后，输入命令jupyter notebook运行，自动打开浏览器页面。
